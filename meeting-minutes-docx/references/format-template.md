# 会议纪要输出模板

严格按以下结构输出，不改变一级/二级标题层级。

```markdown
# 会议纪要：[项目 / 论文 / 主题名称]

## 一、会议主题

**主讲人：** [姓名]

[主讲人姓名] 分享了关于 **[研究方向 / 项目名称 / 论文名称]** 的相关工作。

该研究主要针对 **[核心问题，例如效率、精度、系统设计等]**，通过 **[关键技术 / 方法1]**、**[关键技术 / 方法2]** 等方式，实现 **[主要目标或效果]**。

团队在 **[复现 / 调研 / 实验 / 项目开发]** 的基础上，提出了 **[改进方法 / 新模块 / 新思路]**，例如 **[改进方法名称]**，用于 **[改进目标，如提升性能、降低计算量等]**。

---

## 二、问答与讨论

### 1. [讨论主题1]

- **提问人：** [姓名]
  **问题：** [问题内容或讨论点]

- **回答人：** [姓名]
  **回答：** [回答内容或当前进展]

---

### 2. [讨论主题2]

- **提问人：** [姓名]
  **问题：** [具体技术问题]

- **回答人：** [姓名]
  **回答：** [技术细节 / 当前实现方式 / 是否进行了修改]

---

### 3. [讨论主题3]

- **提问人：** [姓名]
  **问题：** [关于应用、落地或研究背景的问题]

- **回答人：** [姓名]
  **回答：** [项目来源、应用场景或未来可能方向]

---

## 三、总结

本次会议主要围绕 **[核心主题]** 展开，重点讨论了：

- [关键技术点1]
- [关键技术点2]
- [潜在改进方向]

**后续计划：**

- [后续研究方向1]
- [后续研究方向2]
- [实验或实现计划]
```
